# 💰 가계부 애플리케이션 (Money Tracker)

이 프로그램은 개인의 수입, 지출, 투자, 부동산, 대출 등 **모든 자산 현황을 한눈에 관리**할 수 있도록 도와주는 웹 기반 가계부 애플리케이션입니다.

비개발자분들도 쉽게 따라 하실 수 있도록 설치 및 사용 방법을 정리해 드립니다.

---

## 🚀 1. 시작하기 (설치 방법)

이 프로그램을 실행하기 위해서는 컴퓨터에 **Python(파이썬)**이 설치되어 있어야 합니다.

### Step 1: 파이썬 설치
1. [Python 공식 홈페이지](https://www.python.org/downloads/)에서 본인의 운영체제(Windows/Mac)에 맞는 최신 버전을 다운로드하여 설치합니다.
2. 설치 시 반드시 **"Add Python to PATH"** 체크박스를 선택해주세요.

### Step 2: 프로그램 다운로드 및 준비
GitHub에서 소스 코드를 가져오는 방법은 두 가지가 있습니다.

**방법 A: 압축 파일로 받기 (가장 쉬운 방법)**
1. GitHub 페이지 상단의 초록색 **[<> Code]** 버튼을 클릭합니다.
2. **[Download ZIP]**을 선택하여 파일을 다운로드합니다.
3. 다운로드한 압축 파일을 원하는 폴더에 풀고, 해당 폴더로 들어갑니다.

**방법 B: Git 사용하기**
터미널(또는 CMD)에서 아래 명령어를 입력합니다.
```bash
git clone https://github.com/사용자아이디/저장소이름.git
cd 저장소이름
```

4. 해당 폴더의 주소창에 `cmd`를 입력하고 엔터를 눌러 터미널(명령 프롬프트)을 엽니다.

### Step 3: 필수 라이브러리 설치
터미널에 아래 명령어를 복사해서 붙여넣고 엔터를 누릅니다. (프로그램 실행에 필요한 도구들을 설치하는 과정입니다.)
```bash
pip install -r requirements.txt
```

## 🏃 2. 프로그램 실행 방법

모든 준비가 끝났다면, 아래 명령어를 입력하여 프로그램을 실행합니다.

```bash
python app.py
```

명령어를 치면 `Running on http://127.0.0.1:5000` 이라는 문구가 뜹니다.
이제 인터넷 브라우저(크롬, 엣지 등)를 열고 주소창에 아래 주소를 입력하면 가계부가 나타납니다!

> **주소: [http://127.0.0.1:5000](http://127.0.0.1:5000)**

## 📊 3. 주요 기능 및 사용법

상단 메뉴를 통해 각 기능을 이용할 수 있습니다.

### 1️⃣ 대시보드 (Dashboard)
- **자산 총괄**: 현재 나의 순자산, 총자산, 총대출 현황을 그래프와 숫자로 보여줍니다.
- **투자 비중**: 주식, 코인, 부동산 등 자산이 어디에 집중되어 있는지 파악할 수 있습니다.
- **지출 요약**: 이번 달에 돈을 어디에 가장 많이 썼는지 카테고리별로 보여줍니다.

### 2️⃣ 수입/지출 관리
- **수입 (Income)**: 월급, 부수입 등의 내역을 날짜별로 기록합니다.
- **지출 (Budget)**: 매일 발생하는 생활비 지출을 기록하고 관리합니다.
- **카드 관리 (Cards)**: 보유한 카드 정보와 카드별 상세 결제 내역을 관리합니다. (엑셀 파일을 업로드하여 내역을 자동으로 불러올 수도 있습니다!)

### 3️⃣ 투자 및 자산 관리
- **투자 (Investments)**: 주식, ETF, 코인의 매수 가격과 현재 가격을 입력하여 수익률을 실시간으로 확인합니다.
- **부동산 (Real Estate)**: 보유한 부동산의 가치와 임대 수익(월세 등)을 관리합니다.
- **연금 (Pension)**: 국민연금, 개인연금 등 미래를 위한 자산 누적액을 기록합니다.

### 4️⃣ 대출 및 목표 관리
- **대출 (Loans)**: 대출 잔액, 이자율, 상환 예정일 등을 관리하여 상환 계획을 세웁니다.
- **목표 (Goals)**: '1억 모으기' 같은 목표를 설정하고, 현재 얼마나 달성했는지 진행률을 확인합니다.
