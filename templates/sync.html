백업 — JSON 파일로 저장

현재 모든 데이터(주식·ETF·코인·지출·수입·대출 등)를 .json 파일로 다운로드합니다.
구글 로그인 연동 시 이 파일로 데이터를 그대로 복구할 수 있습니다.

복구 — 백업 파일로 덮어쓰기

이전에 저장한 .json 백업 파일을 선택하면 현재 데이터가 전부 교체됩니다.
복구 전 반드시 현재 데이터를 백업하세요.

내보내기 — 이 컴퓨터 데이터를 복사

버튼을 누르면 전체 데이터가 텍스트로 생성됩니다. 전체 복사 후 이메일 본문에 붙여넣어 전송하세요.

가져오기 — 복사한 데이터로 교체

다른 컴퓨터에서 복사한 텍스트를 붙여넣고 가져오기를 누르세요.
현재 데이터가 교체됩니다. (직전 데이터는 자동 백업)

소스 코드를 가져온 후 서버를 재시작해야 변경사항이 적용됩니다.
(터미널에서 Ctrl+C 후 python app.py 재실행)
내보내기 — 이 컴퓨터 소스 코드를 복사

현재 앱의 모든 소스 파일을 텍스트로 묶습니다. 전체 복사 후 이메일 본문에 붙여넣어 전송하세요.

가져오기 — 복사한 소스로 파일 교체

다른 컴퓨터에서 복사한 소스 텍스트를 붙여넣고 비교하기를 누르세요.
가져온 후 서버를 재시작해야 적용됩니다.
